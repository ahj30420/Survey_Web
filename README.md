# Survey_Web

## 한눈에 보는 통합 온라인 설문조사

### 프로젝트 기간
2023.04 ~ 2023.05
### 개발 인원
1명(토이 프로젝트)

* * * 

프로젝트 개발환경은 다음과 같습니다.

#### 개발환경
* JSP, Servlet
* Java, MySql
* Apache Tomcat 9.0

#### 구현 부분
* 사용자 페이지
    - 회원가입 및 로그인(회원 정보 수정)
    - 설문조사 참여 및 포인트 획득
    - 설문조사 분류(전체, 진행, 마감, 완료)
    - 설문조사 제목 검색
    - 퀵폴 생성 / 삭제 / 참여
      
* 관리자 페이지
    - 회원 관리(사용자, 관리자, 문의 내역)
    - 콘텐츠 관리(설문조사, 퀵폴)
    - 결과 분석(설문조사 분석, 퀵폴 분석)
